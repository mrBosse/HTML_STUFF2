abbr & blockquote

  • abbr
    • If you use an Abbreviation, then the abbr title="abbreviation""abbr"/abbr element can be used. A title attribute on the opening tag is used to specify the full term, abbr title ="abbreviation".
  • blockquotes
    • The Blockquote element is used for longer quotes that take up an entire paragraph. You use the the "p" element inside the blockquote element. The qoute will be indented.

acronym

  • acronym
    • If you use an Acronym, then the @"abbr" element can be used. A title attribute on the opening tag is used to specify the full term, abbr title ="United Nations", this will make the wording United Nations appear when hoovring the over the acronym UN.

table & col & colgroup & tr & th

  • table
    • The "table" element is used to create a table. The contents of the table are written out row by row. This is the start and stop of the table.
  • tr
    • The "tr" element stands for table row and You indicate the start and stop of each row. It is followed by one or more "td" elements.
  • td
    • The "td" element stands for table data and represents the cell.
  • th
    • The "th" element stands for table heading and represents the heading for either the column or row. syntax is "th scope="col"Name /th" for column and "th scope="row"Name /th" for row.
  • colgroup
    • The "colgroup" element is used when You want format one o more columns in a table for formatting. It is useful for applying styles to entire columns, instead of repeating the styles for each cell, for each row. Must be a child of a "table" element, after any "caption" elements and before any "th", "tb", "tf" or "tr" elements.
  • col
    • The "col" tag is used to define a different property within the "colgroup".
